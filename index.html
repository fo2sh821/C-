how to run c++ in Visual Studio Code

you must be following this steps

  1. download msys

    download from this here

      write this codes when it launch

    • { pacman -Syu }
    • { pacman -Su }

      • exit

      if letter is capetel or small put it

  2. run MSYS2 MINGW64

      write this codes when it launch

    • { pacman -Ss gcc }
    • search first "mingw-w64-x86_64-gcc" and write { pacman -S mingw-w64-x86_64-gcc }
    • pacman -Ss gdb
    • search first "mingw-w64-x86_64-gdb" and write { pacman -S mingw-w64-x86_64-gdb }
  3. it's last step "Check Versions"

      write this codes

    1. gcc --version
    2. g++ --version
    3. gdb --version

to run c++ in Command Prompt(cmd) following this steps

  1. go to search & write "Edit the system environment variables"
  2. click on advanced
  3. click on environment variables
  4. search in system variables "path" then double click on it
  5. then click new
  6. then add this link on it {c:\msys\mingw64\bin}
  7. then click in "ok" buttons
